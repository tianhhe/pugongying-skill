# KOL Scout 安装说明

## 你需要准备

- Mac 电脑（Windows 未测试）
- [Claude Code](https://claude.ai/code) 已安装
- Chrome 浏览器（普通版即可）
- Python 3.9 或以上

---

## 安装步骤

### 第一步：复制 skill 文件夹

把这整个 `kol-scout` 文件夹放到你的 Claude skills 目录：

```
~/.claude/skills/kol-scout/
```

### 第二步：安装依赖

打开终端，运行：

```bash
python3 ~/.claude/skills/kol-scout/scripts/setup.py install
```

会自动安装：
- `playwright`（浏览器自动化库）
- Chromium 驱动（约100MB，首次安装需要几分钟）

### 第三步：启动 Chrome（开启调试模式）

每次使用前，需要用调试模式启动 Chrome。在终端运行：

**macOS：**
```bash
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --remote-debugging-port=9222 \
  --user-data-dir="/tmp/chrome-kol-scout"
```

> 如果你用的是 "Google Chrome 2"，把路径换成对应的应用路径。

### 第四步：登录账号

在打开的 Chrome 里：
1. 访问 [pgy.xiaohongshu.com](https://pgy.xiaohongshu.com) 登录蒲公英（公司账号）
2. 访问 [xiaohongshu.com](https://www.xiaohongshu.com) 登录小红书（个人账号）

### 第五步：开始使用

打开 Claude Code，输入：

```
/kol-scout
```

Claude 会引导你完成筛选。

---

## 常见问题

**Q：提示"Chrome CDP 端口9222未连接"**  
A：Chrome 没有用调试模式启动，按第三步重新启动。

**Q：提示"蒲公英未登录"**  
A：在 Chrome 里手动访问蒲公英并登录。

**Q：抓不到博主数据**  
A：查看 `~/.claude/skills/kol-scout/tmp/screenshots/` 里的截图，截图能看到页面状态。

**Q：运行很慢，正常吗？**  
A：正常，系统故意放慢速度模拟真人操作，避免被平台风控。100个博主大约需要2-3小时，放后台跑就好。
